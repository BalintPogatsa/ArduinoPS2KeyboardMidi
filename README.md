# ArduinoPS2KeyboardMidi
MIDI Controller from an old PS2 keyboard and an Arduino (I was using Arduino BT-v1, but probably works with Uno and other boards)

Keys are mapped to work with the BP Synth [https://www.youtube.com/watch?v=SV-m3LPFPdk&t=5s]
